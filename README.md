# TestRite-KPG
TestRite King of Popular gathering

提供 OMS 的價格與庫存兩項主力資訊給 user。

APP 功能：<br>
<ol>
  <li>商品掃描
    <ul>
      <li>可用 UPC 或是 QR-code</li>
      <li>掃描結果：店名、庫存、售價</li>
      <li>[集氣吧!]：可以分享到 FB (短時間、長時間)</li>
    </ul>
  </li>
  <li>集氣罐
    <ul>
      <li>關注商品集氣狀態</li>
      <li>使用道具</li>
    </ul>
  </li>  
  <li>店長加持</li>
  若集氣成功就可享有折扣
  <li>道具</li>
  不定時或到店時提供的小道具，可增加集氣量。
</ol>
